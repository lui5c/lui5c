Some websites and programs that I made for myself and the classes that I've taught.

Websites

Number Lines - Useful website for making number lines to practice fractions (3rd grade). Fork the Rel to customize the bounds and subdivisions of each number line.

Bingo Card Maker - A fork of this website for making bingo cards for multiplication practice (3rd grade).

Fraction Generator - a website for creating randomized sets of fractions. Good for practicing ordering, adding, multiplying, etc. Limits for numerator/denominator are possible by forking and making small edits.

Morning Meeting - A website for choosing an animal and then seeing randomized assortment of that animal, for likening oneself to in the morning.

Sierpinski's Triangle - an implementation in Processing.js of a self-generating Sierpinski triangle for triangles that the user can resize (not only equilateral triangles.) User can modify how large the "dots" that populate the triangle are. Bigger dots reveal the pattern faster, smaller dots reveal a more finely-detailed pattern.

Polyrhythm #1 - website for visualizing drum polyrhythms, between kick, hihat, and snare. Instructions on the website.

Post-its - a virtual Post-it implementation with a PHP/SQLite backend. Useful for making seating charts, class groupings, and other tasks that benefit from spatial sorting. Saving session contents is disabled on the Replit implementation. If you fork the code and run your own instance, uncomment the HTTP/POST section of router.php and the synchronize button will work. Your saved Post-its will be loaded into an SQLite database and displayed when you reload the server.

Programs

Conway's Game of Life - a live-editable implementation of Conway's game of life in Java, using Swing. It pauses every 200 generations for you to edit cells.

Gimkit Set Creator - a Python program that can be used to generate CSV files that adhere to Gimkit's requirements for a batch question import file. Set the variables at the top according to whether you would like to practice addition, subtraction, multiplication, or division, and which numbers you would like to practice, run it, and then download the CSV file.

Drum Machine - Another drum machine with better concurrency and timing performance than the website version. Compose instrument patterns by typing in the quarter notes of X's and O's. The variable length patterns for each instrument invites experimentation with polyrhythms.
